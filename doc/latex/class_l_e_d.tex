\doxysubsection{Descripción detallada}
Clase para controlar un \doxylink{class_l_e_d}{LED}. 

Esta clase permite encender, apagar, alternar el estado y hacer parpadear un \doxylink{class_l_e_d}{LED} conectado a un pin digital.
