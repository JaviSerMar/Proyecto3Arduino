\doxysubsection{Descripción detallada}
Clase encargada de publicar mediciones de CO2 y temperatura usando BLE. 

La clase \doxylink{class_publicador}{Publicador} controla la emisora BLE y se encarga de enviar los datos de mediciones como CO2 y temperatura, encapsulándolos en anuncios i\+Beacon.
