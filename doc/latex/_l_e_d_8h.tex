Declaración de la clase \doxylink{class_l_e_d}{LED} y funciones asociadas para controlar un \doxylink{class_l_e_d}{LED} en una placa de desarrollo. 

Este archivo define la clase {\ttfamily \doxylink{class_l_e_d}{LED}} para controlar el encendido, apagado y parpadeo de un \doxylink{class_l_e_d}{LED} conectado a un pin digital de una placa de desarrollo (como Arduino).
